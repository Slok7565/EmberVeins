## 🌌 世界观简介

在名为“星陨”的灾难性事件后，真理星陨落，人类体内觉醒出名为“心源”的神秘能量核心。这是一个关于**宿命与徒劳**、**知识的代价**、**牺牲与救赎**的史诗故事，围绕十二元素体系、神明权柄与心源科技展开，从宏大叙事与平凡人的视角，见证文明的余烬如何在星陨之痕中重燃。
