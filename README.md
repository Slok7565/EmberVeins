## 🌌 世界观简介

在一个经历了"星陨"事件的世界中，人类体内觉醒出名为"心源"的神秘能量核心。这是一个关于牺牲、权柄与文明存续的史诗故事，围绕着十二元素体系、神明权柄与心源技术展开。
