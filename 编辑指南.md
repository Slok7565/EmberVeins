# 《余烬脉流》世界观投稿格式指南

欢迎您为《余烬脉流》世界观创作内容！为保持文档风格统一、便于整理与阅读，请您遵循以下 **Markdown 格式规范** 撰写稿件。

---

## 📝 一、Markdown 基础格式

### 1. 标题层级
使用 `#` 表示标题层级

```markdown
# 标题
## 副标题
### 二级子章节标题
#### 三级子章节标题
##### 四级子章节标题
###### 五级子章节标题
```

### 2. 强调与引用
- **粗体**：`**粗体**`（用于强调关键概念、术语、时间点等）
- *斜体*：`*斜体*`（用于角色名称、时期等）
- ~~删除线~~：`~~删除线~~`
- > 引用块：`> 引用内容`（用于摘录、语录、特殊说明）

### 3. 列表
- 无序列表：`- 项目`
- 有序列表：`1. 项目`
- 嵌套列表：缩进两个空格

示例：
```markdown
- 主要角色
  1. 白明
  2. 卡莱尔·影歌
- 重要地区
  - 曦寰域
  - 奥拉提卡
```

## 🔗 二、链接与引用

### 1. 内部链接（重要！）
请尽量使用 **相对路径** 链接到世界观内其他文档。

```markdown
[编年史](./设定集/核心/编年史.md)
[代号·皑 (Cendre)](./设定集/角色集/角色/代号·皑%20(Cendre).md)
```

相对路径讲解：
- `./`：当前目录
- `../`：上一级目录

### 2. 外部链接
```markdown
[维基百科](https://zh.wikipedia.org)
```

---

## 🖼️ 三、表格与代码块

### 1. 表格
用于角色属性、数据对比等。

```markdown
| 属性 | 说明  |
|----|-----|
| 元素 | 火、光 |
| 权柄 | 焰、虹 |
```

### 2. 代码块
- 普通代码块：用三个反引号包裹
- 指定语言：可在开头的反引号后注明语言（如 `json`, `python`）

````markdown
```python
def example():
    print("Hello World")
```
````

## 📁 四、文件与路径规范
### 1. 文件命名
- 角色文件：`角色名.md`
- 世界观文件：`设定名称.md`
- 事件文件：`事件名.md`

### 2. 目录结构参考
```
EmberVeins/
├── 设定集/         # 设定文档总集
│   ├── 事件/       # 事件设定（神战、创世等）
│   ├── 体系/       # 体系设定（元素、异变体系等）
│   ├── 地区集/地区  # 地理与政体设定
│   ├── 核心/       # 核心设定（编年史、世界观概述等）
│   ├── 角色集/角色  # 角色相关设定
│   └── 项目/       # 系列作品设定
└── 内容集/         # 故事、小说等内容
    ├── 图像内容     # 图片
    ├── 文本内容/    # 小说、短篇等
    └── 音乐内容/    # 音乐
```

## ✨ 五、内容写作建议

### 1. 文档元信息
请在文档结尾注明以下信息：

```markdown
**创作日期**：2026-01-20
**最后更新**：2026-01-25
```

### 2. 时序标注
涉及时间线的部分，请尽量标注 **纪元年份**，并链接到编年史：

```markdown
[远古纪元1716年](./设定集/核心/编年史.md#纪元1716年)
```

### 3. 粗体要求
- **重点内容**：如 本设定库**持续构建中**
- **神明名讳**：如 **坎之神**、**道之神·子午**
- **心源能力**：如 **光蚀刃**、**苍焰·净罪之环**
- **技术名称**：如 **心械复合技术**、**智械逻辑学**
- **组织国家**：如 **空烬团**、**瀚溟仙朝**
- 

### 4. 段落与节奏
- 段落不宜过长，建议 **3-5行** 为一个自然段
- 关键情节或转折可单独成段
- 多使用 **空行** 分隔内容块，增强可读性

---

## 🎯 六、投稿流程

1. **创建文档**：按上述格式撰写内容
2. **内部链接**：确保链接路径正确
3. **预览检查**：在Markdown编辑器中预览效果
4. **提交归档**：按类别放入相应目录
