# 奥拉提卡 (Oratica)

## 大陆概览

奥拉提卡是一片被古老信仰、新生矛盾与扭曲科技树撕裂的大陆。其独特的政治实体——奥拉提卡联邦，在第二次神战后曾带来短暂的秩序曙光，但在星陨纪元前后，这台精密的机器已因内部的信仰纷争与结构性腐败而濒临崩溃。**焰之神**的势力以其特有的侵略性与破坏力，在整个大陆范围内渗透，使得本已复杂的局势更加危殆。大陆的每一个角落，都弥漫着神权与俗权、守护与毁灭、信仰与生存、以及**高端智能**与**落后基础**并存的奇异景象。

## 地理总览

奥拉提卡大陆地形多样。**北部**为严寒的冰川与冻土；**西部**以森林、丘陵与蜿蜒的海岸线为主；**南部**气候温暖，遍布港口与岛屿；**东部**则由连绵的山脉与起伏的丘陵构成；**中部**是广阔的平原与盆地，成为大陆的交通枢纽。大陆四面环海，但落后的航海技术限制了与外界的交流。

## 核心历史脉络与科技悖论

1.  **远古至第二次神战**：各国由各教会神权统治。科技水平整体低下。
2.  **联邦建立与“心源科技”的崛起**：为制衡教会，联邦鼓励世俗研究。星陨纪元后，科学家们偶然发现：“心源”能量（即使在未觉醒者体内也存在微弱的背景辐射）是一种极佳的信息载体和逻辑运算介质。这催生了绕过传统电力与半导体、直接基于心源能量运行的**心源逻辑核心**。
3.  **科技悖论的形成**：
    - **强项（软件与智能）**：奥拉提卡在**分析型AI、复杂系统模拟、信息存储与处理**方面达到了空前高度。联邦和主要国家拥有强大的分析AI来管理数据、辅助研究甚至进行社会管控。
    - **弱项（硬件与基础）**：由于心源能量难以高效转化为大规模的宏观动能，且传统物理学研究被忽视，其在**交通（车辆、飞行器）、军事（枪炮）、能源远程输送**等领域极为落后。大陆的主流运输方式仍是帆船、马车和畜力，战争形态以冷兵器、元素能力与基于心源的防御工事为主。
    - **总结**：奥拉提卡的科技树严重偏向于静态的、网络化的基础设施（如固定轨道、城市建筑、信息网络），而在小型化、可移动的动力单元上近乎空白。
4.  **星陨纪元的影响**：心源觉醒者的大量出现，为心源科技提供了更强大的个体能量源，但并未改变基础科技的落后局面。相反，强大的个体力量反而降低了对发展集体基础科技的需求。
5.  **科技应用的不平衡**：各国对心源科技的态度因其文化和信仰而异，从深度融合到完全排斥。

## 奥拉提卡联邦 (The Oratican Federal)

- **性质**：松散邦联制政体。
- **首都**：**纽枢城 (The Nexus)** - 一座位于大陆中心的人造中立城市。
- **地理位置**：坐落于大陆中央的**奥拉提卡盆地**核心，地势平坦，是所有主要商路的交汇点。
- **成立初衷**：制衡教会权力，协调各国资源，维持大陆和平与稳定。
- **当前状况**：
    - **结构性腐败**：联邦议会沦为各国利益交换的场所，议案常因相互掣肘而无法通过。
    - **权力空心化**：联邦法令在各国境内执行力度极弱，各国常以“主权”和“内部事务”为由拒绝联邦干涉。
    - **信仰角力场**：联邦议会本身已成为各国教会势力（尤其是对抗焰神渗透的联盟与亲焰神势力）博弈的延伸舞台。
    - **军事无力**：联邦常备军规模小，且由各国抽调的部队组成，忠诚度与协调性存疑，无法有效应对跨国危机。
    - **科技依赖与落差**：联邦高度依赖其AI系统进行情报分析和流程管理，但其物理执行能力与成员国相比并无优势。

## 成员国详述

### 1. 诺斯加德 (Norsgard)
- **文化原型**：**北欧诸国**
- **地理位置**：位于大陆**北部**，囊括了广袤的**永冻冰川**、巍峨的**斯卡尔山脉**以及深邃的**冰霜峡湾**。气候严寒，资源以矿产和渔业为主。
- **政体**：名义上为**议会君主制**，实权已被**冰神殿**掌控。
- **主流信仰**：**冰之神**
- **科技态度**：**保守利用型**。仅在**零点实验室**等特定研究设施中集中使用高端科技，民间社会几乎与这类科技绝缘。
- **内部矛盾**：
    - **焰神渗透（最严重）**：焰神使**代号·焚**及其信徒利用底层矿工、对教条不满的年轻战士，宣扬用“烈火”焚尽冰神殿的“冰冷统治”。
    - **人民的苦难**：绝大多数人民在冰神殿的严苛教规与焰神信徒引发的动荡中挣扎求存，对教会极度不信任却又无力反抗。

### 2. 瑞文德尔 (Rivendell)
- **文化原型**：**文艺复兴时期的意大利城邦**
- **地理位置**：位于大陆**西部**，拥有茂密的**翡翠森林**、纵横交错的**水城运河**和曲折的**碎星海岸**。气候温和湿润，贸易与艺术极为发达。
- **政体**：**商人贵族共和议会**，但**岚之吟唱教会**通过掌控经济命脉深度影响政治。
- **主流信仰**：**岚之神**
- **科技态度**：**商业实用型**。商人家族会利用AI进行市场预测、物流管理和金融计算。科技是赚钱和提升影响力的工具。
- **内部矛盾**：
    - **发展 vs 守护**：商人议会主张开凿运河、砍伐森林以发展贸易，与信奉自然平衡的岚之教会冲突激烈。
    - **焰神的阴影**：焰神信徒在激进的工商业主和失地农民中发展势力，鼓吹用“烈火”焚尽贵族议会的“奢靡”与岚之教会的“迂腐”。

### 3. 阿斯塔纳 (Astana)
- **文化原型**：**沙俄&奥斯曼**
- **地理位置**：占据大陆**西南部**的广袤区域，包括荒凉的**赤岩荒漠**、富含矿藏的**铁脊山脉**以及风吹草低的**黄金草原**。气候干燥，昼夜温差大。
- **政体**：**军事集权神权国**，由**岩王殿**与“沙皇”般的军事领袖共同统治。
- **主流信仰**：**岩之神**
- **科技态度**：**军事工具型**。岩王殿和军方会秘密发展用于军事侦察、密码破译和大型工程计算的AI与心源设备。
- **内部矛盾**：
    - **资源枯竭与社会压抑**：过度开采导致主要矿脉濒临枯竭。岩王殿的严格等级制度与军事压迫引发底层矿工和边境部落的不满。
    - **焰神的蛊惑**：焰神势力在贫困的矿工和渴望军功与变革的年轻军官中渗透，宣扬用毁灭性的力量重塑秩序。

### 4. 虹霞圣土 (Iridia)
- **文化原型**：**封建日本**
- **地理位置**：位于大陆**东部**，由**霞光群岛**与**本州半岛**组成，境内多山（如**神圣的浮金山**），沿海有狭长平原。以春季绚烂的樱花和秋季漫山的红叶闻名。
- **政体**：**双教宗共治**，由**光耀教廷**（天皇般的象征与精神领袖）与**暗影秘会**（幕府将军般的实际掌权者）共同治理，极度不稳定。
- **主流信仰**：**光之神**与**影之神**（信仰分裂）。
- **科技态度**：**分裂对立型**。**光耀教廷**倾向于使用AI进行社会监控和“维护正统”。**暗影秘会**则视科技为亵渎，推崇传统技艺与人工情报网络。
- **内部矛盾**：
    - **光与影的内战**：光与影教会及其支持的封建大名之间的冲突连绵不断，形式从宫廷阴谋到武士战争。
    - **焰神的渔利**：焰神势力在光与影势力的夹缝中生存，专门吸纳对无休止内斗和严苛礼法感到绝望的浪人与平民。

### 5. 时计领 (The Chronarchy)
- **文化原型**：**维多利亚时期英国**
- **地理位置**：位于大陆**中部**，环绕联邦首都的**波伦特平原**。地势低平，河网密布，拥有大陆最完善的**心源铁道**网络（以心源驱动的机械轨道车）。
- **政体**：**神权政治**，由**时之律法院**通过庞大的官僚系统和时钟塔网络实行严格管理。
- **主流信仰**：**时之神**（崇尚精确、秩序、进步与**被打破的既定命运**）。
- **科技态度**：**深度融合型**。时计领是心源科技与社会结合最深的国度。**时之律法院**通过遍布全国的“钟塔网络”和AI系统，对社会进行精确到秒的管理，并试图**重新计算和固化**时序。
- **内部矛盾**：
    - **控制的悖论**：时之律法院深知**子午**已撕裂权柄、打破了唯一的既定命运，但他们坚信通过极致的控制与计算，能在混沌中重新建立“最优化的命运轨迹”。这种试图用秩序束缚已然获得自由的时间的行为，引发了社会各界的反抗，组织“破序者”认为这是对自由意志的亵渎。
    - **焰神的虚无**：焰神信徒视时之律法院的一切努力为徒劳，认为在一个没有既定命运的世界里，唯有彻底的毁灭才能带来真正自由的“新生”。他们通过破坏钟塔网络和AI节点，来证明“命运无法被计算，只能被燃烧”。

### 6. 圣霆要塞 (Sanctified Keep)
- **文化原型**：**十字军国家/条顿骑士团**
- **地理位置**：位于大陆**东南部**的**托尼特鲁斯半岛**，三面环海，背靠险峻的**萨库斯山脉**。半岛终年有雷暴天气，被视为霆之神的恩赐与试炼。
- **政体**：**军事修会统治**，政教完全合一，由**圣雷审判庭**的大审判官们集体领导。
- **主流信仰**：**霆之神**
- **科技态度**：**排斥抵制型**。霆教会将过度依赖AI与心源科技视为对神明权柄的僭越和一种“软弱”。他们坚信苦修、信仰与纯粹的肉体和元素力量才是正道。
- **内部矛盾**：
    - **极端排外与内部清洗**：霆教会视其他所有信仰（尤其是焰之神）为必须净化的异端。审判庭内部因对“净化”尺度理解不同，内斗不断。
    - **焰神的死敌**：作为秩序与净化的极端代表，圣霆要塞是焰之神势力渗透最困难、但也是其最想摧毁的目标。

## 总结：撕裂与并存的世界

奥拉提卡是一个在科技上严重“偏科”的世界。人们可以创造出拥有接近自我意识的人工智能，却无法造出一辆汽车；可以利用心源进行超距通讯，军队的主力却仍是手持附魔刀剑的士兵。这种独特的科技悖论，深深根植于“心源”这一特殊能量源的性质，也反映了各国文化与信仰的差异。

如今，这股强大的智能科技力量，与复苏的神权、渗透的毁灭信仰交织在一起，使得奥拉提卡的未来变得更加扑朔迷离。无论是AI推算出的命运，还是神明昭示的预言，似乎都指向了一个充满烈焰与变革的终局。联邦的松散、各国的内忧外患、以及焰之神在**诺斯加德**点燃的火线，正将整个大陆推向全面战争的边缘。
