# 三次神战详述

## 概述
三次神战是贯穿《余烬脉流》远古至星陨纪元的核心历史事件，其本质是 **对世界法则支配权（权柄）的争夺、理念冲突与时代更迭**。战争导致神明陨落、权柄破碎、元素分化，并最终为凡人与心源力量登上历史舞台铺平道路。

## 第一次神战：元素王座的倾覆

- **时间**：远古纪元1708年 – 1742年
- **核心矛盾**：**原初神明间长期存在的理念摩擦与势力失衡**。虹之神并非战争的主动发起者，但其过于强大的力量与对“完美”的偏执追求，使其成为其他神明忌惮与围攻的目标。
- **直接导火索**：在一次诸神集会上，虹之神展示其新领悟的“万色归一”之境，其力量无意识地覆盖并短暂压制了其他元素的辉光。此举被**岚之神**视为对其概念的终极挑衅与威胁，遂联合对虹之神早有不满的**焰之神**，秘密策划了弑神行动。

### 阵营与各方立场
| 阵营         | 主要成员                       | 立场与动机                                                                                                                            |
|:-----------|:---------------------------|:---------------------------------------------------------------------------------------------------------------------------------|
| **“颠覆”联盟** | **岚之神** (主导)、**焰之神** (主力)  | **岚之神**：视虹之神为“绝对秩序”的化身，是自由的最大敌人。旨在消除这一威胁，并意图在战后新秩序中获得主导地位。<br>**焰之神**：渴望变革，厌烦现状，认为虹之神是旧秩序最坚固的象征。与岚之神结盟以获得颠覆性的力量。                 |
| **“维护”阵营** | **虹之神**、**岩之神** (后期)       | **虹之神**：最初仅为展示力量，并无主动侵略意图。战争爆发后转为自卫与反击，试图维护其作为最古老、最强大神明的尊严与地位。<br>**岩之神**：起初中立，但在目睹战争对世界物质基础的破坏后，认为“颠覆”联盟的破坏性更大，选择站维护基本秩序，支持虹之神。 |
| **中立/观望方** | **坎之神**、**霆之神**、**道之神·子午** | **坎之神**：主张调和，但双方均不愿接受调停，被迫保持距离，守护自身领域内的生命。<br>**霆之神**：内部对谁是“秩序破坏者”争论不休，举棋不定。<br>**道之神·子午**：以全知之眼冷眼旁观，维系时空结构，不介入具体纷争。             |

### 战争经过与结果
- **经过**：战争始于岚与焰之神对虹之神领域的突袭。虹之神凭借其压倒性的力量在初期稳住阵脚，甚至一度反击。岚之神利用其无处不在的特性进行骚扰与分割，焰之神则以极致的破坏力主攻。岩之神的加入使战争陷入僵持。战争波及无数早期文明，**虹霞圣土**的毁灭成为战争残酷性的缩影。
- **关键转折与结局**：在僵持阶段，岚之神策划了一次精准的突袭。祂利用一场席卷诸多元素位面的“大寂灭风暴”（本质是抽取了巨量的岚元素，制造出近乎绝对虚无、连光都无法存续的领域），将虹之神短暂地困入其中。焰之神抓住机会，将全部的毁灭性力量注入这片“虚无牢笼”。**虹之神在光暗对冲与极致虚无中，权柄核心破碎，最终陨落。**
- **结果**：
    1.  **虹之神陨落，权柄分化**：其权柄破碎，分化为 **光** 与 **影** 两种新元素权柄。
    2.  **新势力崛起**：岚之神与焰之神成为神战中最大的获益者，影响力大增。*代号·皑* 于此时接受部分虹之权柄碎片，成为早期神使。
    3.  **格局初定**：首次明确了神明间通过联盟与战争可以改变力量格局，猜忌与制衡成为神明关系的新常态。
    4.  **子午的警示**：战后，子午向众神展示了其预见的、通往更黑暗终局的“唯一未来”，但未能引起足够重视。

## 第二次神战：宿命之笼的破碎

- **时间**：远古纪元1975年 – 1996年
- **核心矛盾**：**如何应对子午预见的“天坠”终局**。理念分歧彻底撕裂了神明阵营，形成了以 **“抗争”** 与 **“顺从/利用”** 为核心的对立联盟。
- **直接导火索**：子午预见的“终极虚无”日益清晰。以**焰之神**（融合了部分岚之神的激进思想）为首的联盟，主张主动引发剧变，在毁灭中寻找超越预言的新生；而以**坎之神**、**霆之神**为首的联盟，主张强化现有秩序，规避或“修正”命运。

### 阵营与各方立场
| 阵营                  | 主要成员                                      | 立场与动机                                                                                                                                   |
|:--------------------|:------------------------------------------|:----------------------------------------------------------------------------------------------------------------------------------------|
| **“焚纪”联盟** (主战/变革派) | **焰之神** (核心)、**岚之神** (重要盟友)、**部分新生影之神势力** | **焰之神**：坚信预言中的终局意味着现有一切的彻底无意义，唯有主动的、彻底的毁灭才能焚烧出新的可能性。<br>**岚之神**：支持打破一切束缚，包括“既定的命运”，认为这是自由的终极体现。<br>**影之神势力**：渴望颠覆光之神主导的现有秩序，从混乱中获利。     |
| **“恒常”联盟** (守护/秩序派) | **坎之神** (核心)、**霆之神** (核心)、**岩之神**、**光之神** | **坎之神**：主张守护生命与文明的存续，认为通过调整与强化可以找到生路，反对无差别的毁灭。<br>**霆之神**：视“焚纪”联盟为必须净化的“混沌与亵渎”，坚信神圣秩序能抵御命运。<br>**岩之神** & **光之神**：倾向于维护现有世界的稳定结构，加入守护阵营。 |
| **超然时序**            | **道之神·子午**及其神使                            | **子午**：洞悉所有抗争在既定命运下皆为徒劳。其目标非击败任何一方，而是**打破命运本身**。战争是其计划的舞台与掩护。                                                                           |

### 战争经过与结果
- **经过**：战争在两大联盟间全面爆发，规模与烈度远超第一次。焰之神的毁灭烈焰、岚之神的无形侵蚀对阵坎之神的生命守护、霆之神的绝对裁决。世界多处化为焦土与废墟。
- **关键转折 - 子午的牺牲**：战况最激烈时，子午目睹挚友**坎之神**在守护中力竭陨落。这一事件成为最终催化剂。**子午抓住坎之神权柄破碎、时空剧烈震荡的瞬间，调动全部力量，决绝地撕裂了自身的“道”之权柄。**
- **结果**：
    1.  **双重陨落**：**坎之神**权柄破碎分化为 **水** 与 **冰**。**道之神·子午**权柄破碎分化为 **时** 与 **空**。
    2.  **宿命打破**：时空根基的撕裂，强行中断了战争，更从根本上动摇了唯一的宿命之河，世界进入“可能性”时代。
    3.  **权柄继承与异变**：
        - *辰序* 与 *纬方* 继承时、空权柄，着手修复时空。
        - *???* 因目睹全过程，受存在主义冲击，自我抹除，成为悖论幽灵。
    4.  **战争强制终结**：幸存神明在时空混乱的威胁下停战，“焚纪”与“恒常”联盟名存实亡，世界进入恢复期。

## 第三次神战：凡人的黎明

- **时间**：星陨纪元约250年 – 370年
- **核心矛盾**：**凡人对神权暴政的反抗**与 **对心源时代主导权的争夺**。
- **直接导火索**：**霆之神**在其直属国度施行极端严苛的“净化律法”，对稍有异议者、甚至只是心源觉醒形态“不纯”者施以残酷审判。长期积累的恐惧与怨恨，在几位目睹亲友被无故裁决的凡人强者心中点燃了 **“弑神”** 的火焰。

### 阵营与各方立场
| 阵营                 | 主要成员                                                                    | 立场与动机                                                                             |
|:-------------------|:------------------------------------------------------------------------|:----------------------------------------------------------------------------------|
| **“审判庭”阵营**        | **霆之神**及其直属教会（圣雷审判庭）                                                    | **霆之神**：坚信唯有绝对秩序与净化才能让世界免于混乱。视凡人的反抗与心源的不可控性为最大亵渎，意图以铁腕清除。                         |
| **“薪火”联盟** (凡人反抗军) | **三位最初的凡人领袖** (后称“弑神者”)、众多受压迫的心源觉醒者、部分对审判庭不满的霆之神底层信徒、其他元素教会中同情者提供的有限援助。 | **核心诉求**：推翻霆之神的暴政，争取凡人与心源觉醒者自主生存的权利。<br>**理念**：神权不应是无情的压迫，力量应服务于生命与自由。            |
| **其他神明势力**         | **焰之神**、**岚之神**残余势力、**水、冰、光、影**等神系                                      | **态度复杂**：多数乐见霆之神这一“顽固秩序派”受损，但不愿直接与神权为敌，多采取暗中提供线索、或放任不管的态度。焰之神势力甚至暗中鼓励这种“毁灭旧神”的行为。 |

### 战争经过与结果
- **经过**：战争始于一系列针对审判庭设施的秘密破坏与游击。反抗军力量薄弱，正面抗衡无异于送死。他们利用心源能力的多样性进行渗透、情报收集和定点清除。最关键的行动是，他们从古老遗迹和未知存在散落的呓语中，拼凑出了 **“权柄并非绝对无敌，其与神明存在的连接点存在理论上的脆弱环节”** 的禁忌知识。
- **高潮 - 弑神之战**：经过精心策划，三位最强的凡人领袖（一位拥有极高能量操控精度的心源觉醒者，一位精通古老禁术的学者，一位曾被审判庭摧毁一切、心怀死志的前圣殿武士）率领被压迫的人们，在圣霆要塞的核心圣域，利用一场盛大审判仪式汇聚的巨量雷元素作为掩护和能量源，发动了突袭。
    1.  **学者**以自身生命与心源为代价，启动了干扰神明与权柄连接的古老禁术。
    2.  **前武士**正面承受了霆之神的盛怒一击，以灰飞烟灭为代价，为同伴创造了唯一的机会。
    3.  **能量操控者**在连接被干扰的瞬间，将汇聚的、以及自身全部的心源能量，转化为一束超越物质与能量范畴的“纯粹湮灭脉冲”，贯入了霆之神权柄的瞬时裂隙。
- **结果**：
    1.  **霆之神陨落**：神明在错愕与愤怒中崩解，其完整的霆之权柄破碎。
    2.  **权柄瓜分**：三位凡人领袖中仅存的 **能量操控者**，在弑神瞬间与逸散的权柄碎片产生共鸣，因其纯粹的能量特质，吸收了 **明雷 (Bright Lightning)** 权柄碎片。其余两块碎片 **寂雷 (Dread Lightning)** 与 **圣雷 (Sacred Lightning)** 则飞散，后被不同势力或个体获取（成为后续神使或圣物核心）。
    3.  **时代象征**：此战被后世称为“黎明的代价”。它庄严宣告：**凡人凭借智慧、勇气、牺牲与心源的力量，足以挑战并终结旧神。** 神权不可侵犯的时代彻底落幕。
    4.  **连锁反应**：弑神成功极大地鼓舞了奥拉提卡各地对暴政（包括焰之神等其他神明的极端势力）的反抗，加速了焰之神在纪元370年的陨落，并迫使所有残存的神明势力以更谨慎、更间接的方式介入凡世。

## 总结：神战的意义
三次神战构成了力量与秩序迭代的三级阶梯：
1.  **第一次**：**神明时代的内部洗牌**。原初格局打破，元素分化，联盟政治成为常态。
2.  **第二次**：**命运层面的解放**。子午的牺牲为世界争取了自由的“可能性”，代价是至高神权的退场与时空法则的永久改变。
3.  **第三次**：**凡人权力的奠基**。以凡人之躯弑神成功，证明了心源时代的新力量足以颠覆旧秩序，奠定了星陨纪元后期乃至后星陨纪元 **“神权隐退，凡人与心源为主导”** 的世界新基调。
